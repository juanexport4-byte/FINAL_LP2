## 1. Descripción del Problema
En el ecosistema digital actual, el acceso masivo a Modelos de Lenguaje (LLMs) ha abaratado drásticamente el costo de generar contenido falso hiperrealista. Este proyecto aborda la proliferación de **estafas financieras y desinformación** automatizada (Criptomonedas, Phishing, Inversiones Falsas) mediante un sistema automatizado de recopilación de datos.
